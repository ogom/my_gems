An example of Fizz Buzz.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'my_gems'
```

And then execute:

```
$ bundle
```

Or install it yourself as:

```
$ gem install my_gems
```

## Usage

```ruby
fizz_buzz = MyGems::FizzBuzz.new(3)
fizz_buzz.say # "Fizz"
```
